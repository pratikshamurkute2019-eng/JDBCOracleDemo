# Task-1 JDBC Oracle 10g Connection Demo

##  Project Overview
This project demonstrates a **basic JDBC connection between Java and Oracle 10g database**.
It verifies successful database connectivity before implementing CRUD operations.

This task is commonly used as a **first step in real-world JDBC projects**.

##  Technologies Used
- Java (JDK)
- Oracle 10g Database
- JDBC
- IntelliJ IDEA
- Git & GitHub

##  JDBC Connection Steps
1. Load Oracle JDBC Driver using `Class.forName()`
2. Establish connection using `DriverManager.getConnection()`
3. Verify connection status
4. Print **"Connection Successful!"**

##  How to Run
1. Open project in IntelliJ IDEA
2. Add `ojdbc14.jar` to project libraries
3. Update database username and password
4. Run `OracleConnectionTest.java`
